Usage

Quotes are implemented with the class cl-quote

<div class="cl-quote"> <blockquote>The quote</blockquote> <cite>Citation</cite> </div>
Potent quotables are a terrible idea.
No one ever

To add an image, use a div with the class cl-quote-image, and then set a background image using a style attribute. Note that the image will be cropped into a circle.

<div class="cl-quote"> <div class="cl-quote-image" style="background-image:url(someimage.jpg)"></div> ... </div>
Potent quotables look even better when an image is used.
Everyone

Shortcode Support

Quotes can be implemented with the shortcode cl-quote

[cl-quote quote="A great quote" citation="Who said it"]

Attributes

  • quote (string) (required)

    The quote

  • citation (string) (required)

    The citation

  • img (string) (optional)

    The URL of an image

    (default value: none)

  • alt (string) (optional)

    Alt text for the image (will be used as the title for div.cl-quote-image)

    (default value: none)

  • class (string) (optional)

    Set custom CSS class(es)

    (default value: none)
