Components

A website is built using a relatively small number of repeating elements, like buttons, cards, and heros. The Component Library exists in order to standardize the look and feel of these elements, and make it fast and simple to build webpages that look great and stay on-brand.

  • BoxoutsGroup multiple action links or buttons together, with some brief explanatory text.
  • ButtonsFocus user attention on and allow them to take major actions.
  • CardsAn elevated, clickable container with abridged contents and a single action.
  • HerosLarge, full-width components used to display visually engaging content.
  • MenusA navigation framework.
  • NoticesFull-width banners that display a short amount of temporary, time-sensitive content.
  • PanelsLarge content containers constructed of two panes: one displaying text and the other displaying an image.
  • SlideshowsTransform native WordPress image galleries into a user-controllable carousel with a clean UI.
  • Social BarsDisplay a row of social media icons.
  • TabsA user-controlled interface that displays one section of content at a time.
  • VideosCustom YouTube embeds that have minimal YouTube branding and do not show video annotations.
