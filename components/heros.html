Heros

Heros are large, full-width components used to display visually engaging content, typically at the top of the page, and are often accompanied by header language and occasionally a button. Heros can be either a video or a still image, and are used to add visual intensity to a page with the goal of driving the user to action.

Video Hero

Video heros are implemented with the class cl-hero

Video heros require an overlay and poster image. Create an overlay with the class overlay, and a poster with the class poster.

The poster must also have an id attribute. It can be arbitrary, but it is necessary for YouTube's API to function.

The poster's data-id attribute is used to determine which YouTube video to load, based on the video id. For example, if the YouTube URL is https://www.youtube.com/watch?v=ydD1Wg2B8Vk, create the poster with a data-id of ydD1Wg2B8Vk.

<div class="cl-hero"> <div class="overlay"></div> <div id="hero1" data-id="[video id]" class="poster" style="background-image:url(poster.jpg)"></div> </div>

In its most basic form, the overlay prevents any user interaction with the video. The overlay <div> can be empty, or it can contain a block of content as described below. The poster is a placeholder for the video in the event that the video cannot be loaded. The poster hidden if the video loads successfully, but remains before the video in the DOM.

Add overlay content.

To overlay content, such as a headline, subheading, and button, create a content block inside the overlay using the class block. The content will appear superimposed on the video.

<div class="cl-hero"> <div class="overlay"> <div class="block"> <h1>Headline</h1> <p>Subheading</p> <a class="cl-button" href="#">Button</a> </div> </div> <div id="hero1" data-id="[video id]" class="poster" style="background-image:url(poster.jpg)"></div> </div>

Image Hero

Image heros are implemented with the class cl-hero

<div class="cl-hero"> <div class="still" style="background-image:url(hero.jpg)"></div> </div>

Add overlay content.

To overlay content, such as a headline, subheading, and button, create a content block before the image using the class block. The content will appear superimposed on the image.

Note that an overlay div is not needed for image heros.

<div class="cl-hero"> <div class="block"> <h1>Headline</h1> <p>Subheading</p> <a class="cl-button" href="#">Button</a> </div> <div class="still" style="background-image:url(hero.jpg)"></div> </div>

Image heros can be animated.

Heros are designed so that the image is initially centered and always covers the entire hero element. Because of this, the direction and distance of any motion applied is dependent on the dimensions of the image and the hero.

To add movement to the hero image on page load, first add the class animate to the image. Then choose an animation method.

The "shift" method

To use the shift animation, add the class shift. When the page is loaded, the image will shift position slightly.

<div class="cl-hero"> ... <div class="animate shift" style="background-image:url(hero.jpg)"></div> </div>

If the image is wider than the hero, the image will shift left until the right edge of the image is aligned with the right edge of the hero. If the image is taller than the hero, the image will shift down until the top edge of the image is aligned with the top edge of the hero. The animation lasts for 10 seconds.

Options

Heros can be full width.

To make a video or image hero span the width of the browser window, add the class fullwidth

<div class="cl-hero fullwidth"> ... </div>

Shortcode Support

Heros can be implemented with the shortcode cl-hero

[cl-hero headline="A Great Hero" id="hero1" vid="aK9iLeL5l_c" img="sampleimage.jpg"]

Attributes

  • img (string) (required)

    The source URL for the hero image or, if vid is set, the poster image.

  • vid (string) (optional)

    Set the YouTube video ID to create a video hero; id must also be set.

    (default value: none)

  • id (string) (optional)

    Set a unique id for the element; required if vid is set.

  • format (string) (optional) (values: super, fullwidth)

    Specify the hero format

    (default value: none)

  • animation (str) (optional) (values: slide)

    Set to animate an image hero; this will be ignored if vid is set.

    (default value: none)

  • headline (string) (optional)

    The text for a headline

    (default value: none)

  • subhead (string) (optional)

    The text for a subhead

    (default value: none)

  • link (string) (optional)

    Set a URL to create a button.

    (default value: none)

  • button (string) (optional)

    The text for the button

    (default value: 'Explore')

  • tooltip (string) (optional)

    Set a tooltip for the button

    (default value: none)

  • class (string) (optional)

    Set custom CSS class(es)

    (default value: none)

Deprecated

  • super (bool) (optional)

    2.0.0Use the format attribute instead

  • fullwidth (bool) (optional)

    2.0.0Use the format attribute instead

  • dynamic (bool) (optional)

    2.0.0Use the animation attribute instead

WYSIWYG Editor Support

With the WYSIWYG plugin installed, heros can be added using the WYSIWYG editor.

The icon to add a hero appears in the WYSIWYG toolbar. Clicking this icon opens a dialogue where different hero attributes can be easily set.

Currently, superheros and full width heros cannot be built through the WYSIWIG editor.

The completed hero will appear fully styled in the editor window. To edit it, simply double-click the hero to reopen the editor dialogue.
