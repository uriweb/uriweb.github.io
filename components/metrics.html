Metrics

A component that displays a number and a brief caption

  • Usage

Usage

Metrics are implemented with the class cl-metric

<div class="cl-metric"> <span>number</span> <span>caption</span> </div>
55% Rhode Island residents

By default, metrics have a light grey background. To display the metric using a dark background, add the class dark.

To display the metric without a background, for use in cases where the page or parent element background should show through, use the class clear.

To display a clear metric in cases where the page or parent background is dark, use the dark and clear classes together.

<div class="cl-metric dark"> <span>number</span> <span>caption</span> </div>
78% classes with less than 30 students
<div class="cl-metric clear"> <span>number</span> <span>caption</span> </div>
3,298 first-year students
<div class="cl-metric dark clear"> <span>number</span> <span>caption</span> </div>
17:1 student to faculty ratio

Metrics can be floated

Metrics assume the width of their parent by default.

To float the metric, add the class left or right. When floated, a metric's width adapts to fit its contents.

<div class="cl-metric left"> ... </div>
100% to the left

Shortcode Support

Metrics can be implemented with the shortcode cl-metric

[cl-metric metric="78%" caption="classes with less than 30 students"]

Attributes

  • metric (string) (required)

    The text for the metric

  • caption (string) (required)

    The text for the caption

  • style (string) (optional) (values: dark, clear, overlay)

    Set the style of the metric

    (default value: none)

  • float (string) (optional) (values: 'left', 'right')

    Set to float the metric either left or right

    (default value: none)

  • class (string) (optional)

    Set custom CSS class(es)

    (default value: none)
