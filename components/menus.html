Menus

Menus are a navigation framework that transform native WordPress menus.

  • Usage

Usage

Menus are implemented by creating a WordPress menu and then adding it with the shortcode cl-menu

[cl-menu name="awesome-menu"]

Attributes

name (string) (required)

The name of the menu to use
(default value: menu-1)

depth (num) (optional)

Set how many levels of hierarchy to include
(values: 1, 2) (default value: 1)

title (string) (optional)

Set a custom menu title (the menu name is used by default)
(default value: none)

showtitle (bool) (optional)

Set show the menu title on desktop
(default value: false)

id (string) (optional)

Set an ID for the menu container
(default value: none)

class (string) (optional)

Set custom CSS class(es)
(default value: none)
